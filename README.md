The "Duck" then reads their specific plan's coverage and benefits declaration, providing a clear answer on whether the care is covered and outlining any steps needed to access that coverage.

## Installation Instructions
Install Python 3
Download and extract the project
Edit 'main.py' with your OpenAI API token
Open a terminal and navigate to the project
Create an environment 'py -m venv venv'
Activate the environment '.\venv\Scripts\activate'
Install flask 'pip install flask'
Install openai 'pip install openai'
Install pdfPlumber 'pip install pdfplumber'
Install python-dotenv 'pip install python-dotenv'
Run flask 'flask run -h localhost -p 80'
Navigate to localhost on your preferred browser
